Секция 03 · Ценовой движок

Как считается
справедливая цена.

# Шаг 1: рыночная база avg_m2 = SQLite(district, rooms, area ±30%, IQR фильтр) # Шаг 2: корректировки год: до 1950 → −35% 2020+ → +8% этаж: 1-й → −5% последний → −3% тип: панель → −10% монолит → +5% фото: 1-2 → −15% 9-10 → +10% # Шаг 3: диапазон ±8% fair = avg_m2 × (1 + adj%) × area range = [fair × 0.92, fair × 1.08]
Почему алгоритм, а не LLM

Детерминированный код = предсказуемость. Каждый коэффициент задокументирован. LLM мог бы галлюцинировать числа.

Почему ±30% по площади

60м² сравнивается только с 42–78м², не со всеми 2-комнатными. Без фильтра медиана искажается крайними значениями.

Почему IQR

Убирает нотариальные (1₸) и ошибки парсинга. Медиана устойчива к выбросам — нотариальные сделки не искажают рынок.

·Все 10 фото из объявления в одном запросе к GPT-4o
·Жёсткие якоря в промпте — модель не ставит 7–8 "по умолчанию"
·Флаг low_confidence для тёмных/размытых фото — не штрафуем
·Корректировка % — детерминированная функция, не модель
Без Vision

Продавец пишет "хороший ремонт". Агент верит → вердикт "выгодно".

С Vision

Агент видит советские батареи и штукатурку → score=3 → −8% → вердикт "дорого".

Промпт v1 → v2

v1: модель ставила 7–8 почти всем. Исправление: якоря шкалы, убрали освещённость, % в детерминированный код.

Секция 04 · A/B Тест

Оправдывает ли GPT-4o
10× рост стоимости?

Гипотеза

GPT-4o для рекомендации даёт значительно лучший LLM-judge score и оправдывает 10× рост стоимости.

МетрикаConfig A · gpt-4o-miniConfig B · gpt-4o
Verdict Accuracy86.7%86.7%
Relevance Score0.851 ↑0.794
LLM Judge4.97/5 ↑4.90/5
Стоимость/запрос~$0.01~$0.10
Вывод: гипотеза не подтвердилась

Accuracy одинакова — вердикт детерминирован алгоритмом, не моделью. Mini не хуже gpt-4o при цене в 10× меньше.

Решение

GPT-4o остаётся для Vision — замены нет. Рекомендации переходят на gpt-4o-mini — экономия 5–10×.

4 ошибочных вердикта

Все 4 — граничные "рыночная цена", одинаковые в обоих конфигах. Ошибка алгоритма, не модели.

Гипотеза

Реальность

A/B тест опроверг: accuracy одинакова, relevance у mini выше.

Решение

Mini для текста, gpt-4o только для Vision.

Решение

Убрали освещённость, добавили low_confidence флаг.
